# ws.dto

Web/API DTOs used by controllers for requests and responses.

## DTO groups
- BindingDtos
- EntryDtos
- EntryDetailDtos
- MediaDtos
- MediaEntryDtos
- PersonDtos
- PersonEntryDtos
- PersonReadDtos
- PersonRelationDtos
- TagDtos

## Usage
- Used by ws.controller layer and mapped by ws.mapper.WsMapper.

## Public interface pattern
- DTO groups typically include Create/Update/Detail variants.
